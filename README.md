# 💍 Свадебный лендинг — Алексей & Анастасия

Одностраничный сайт-приглашение на свадьбу с формой подтверждения участия (RSVP).

Гости заходят на сайт, знакомятся с программой, смотрят фотографии и подтверждают своё присутствие через форму. Ответы автоматически сохраняются в Google Таблицу — без сервера и базы данных.

---

## 🛠 Стек

- **Vite** — сборщик
- **React 19** — интерфейс
- **TypeScript** — типизация
- **SCSS (Sass)** — стили
- **Google Apps Script** — приём RSVP-ответов в Google Таблицу
- **БЭМ** — методология именования классов

---

## ⚙️ Переменные окружения

Все ключевые данные вынесены в файл `.env` в корне проекта. Чтобы изменить имена, дату или адрес — достаточно отредактировать этот файл, не трогая код.

| Переменная               | Назначение                      |
| ------------------------ | ------------------------------- |
| `VITE_WEDDING_DATE`      | Дата и время свадьбы (ISO 8601) |
| `VITE_GROOM_NAME`        | Имя жениха                      |
| `VITE_BRIDE_NAME`        | Имя невесты                     |
| `VITE_VENUE_NAME`        | Название площадки               |
| `VITE_VENUE_ADDRESS`     | Адрес площадки                  |
| `VITE_RSVP_DEADLINE`     | Крайняя дата подтверждения      |
| `VITE_GOOGLE_SCRIPT_URL` | URL веб-приложения Apps Script  |
